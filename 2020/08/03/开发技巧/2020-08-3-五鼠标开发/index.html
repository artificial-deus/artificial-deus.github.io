无鼠标开发

鼠标和图形化会让我们寻找要做的事情，这会打断心流(flow)。

我们在工作中可以正念，寻找并且关注那些会使我们分心的事情，使用鼠标无时无刻不在影响我们的注意力。

idea设置

  1. 打开并选择可以选择在TOP或底部。

    Ctrl+Shift+A

    Show Breadcrumbs

    打开并选择可以选择在TOP或底部。

  2. 打开面包屑导航

    双击Shift

    Show Navigation Bar

  3. 关闭已打开标签

    Tab placement

    None

  4. 关闭面包屑导航

    Show Navigation Bar

  5. 关闭工具栏

    Tool Window Bars
