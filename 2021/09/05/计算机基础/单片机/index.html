1.1 计算机基本概念

冯诺依曼结构 EDVAC

1946年由冯诺依曼提出的通用电子计算机方案。

  • 算术逻辑部件 ALU Arithmeti Logic Unit

    包括算术计算器与通用寄存器

  • 控制器CU Control Unit

    工作内容包括从内存MEME取指令、分析指令、执行指令。

    它由指令计数器、指令寄存器、译码器及控制逻辑等电路构成。

  • 存储器MEMory

    • 内存
    • 外存储器
      • 属于输入输出设备I/O

  • 输入设备 IU Input Unit

  • 输出设备OU Output Unit
